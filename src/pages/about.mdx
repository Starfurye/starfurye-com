## 自我介绍

在读 CS 研究生，喜欢电子音乐（特别是芯片音乐），轻度发烧友；喜欢像素风格，RPG 游戏爱好者；前端爱好者。

聊聊我和计算机的缘分吧。高一就开始学习编程，当时跟着老师用 Free Pascal 写题。我每周都会有一次晚自习捧着厚重的一本盗版书去机房对着蓝底黄字的 IDE 敲代码。但由于学校在小县城，对传统的学科抓得比较紧，而且对于信息技术这门学科的培养缺乏经验和实力，考虑到学业，结果后面也没去参加比赛，但也算是对写代码有了一个比较粗浅的体验，`program`, `begin`, `end` 还有赋值符号 `:=` 也让我对编程语言有了一点认识。

本科最初的专业电子信息类（大类，要大二才划分具体专业），大一的时候学了 C++，当时还加入了学院的团委新闻中心技术部，做公众号文章排版和编辑工作。现在想来那些排版工作或许就为我埋下了热爱网页设计和布局的种子。大一寒假时还买了单片机开发板，用当时学院教的 C 语言来编写程序，然后烧录到开发板上运行。后来又转去学 python，学习网络爬虫。也许是因为享受写代码的感觉吧，受这股无形的力量推动，大一下学期我向学院申请了转专业，如愿转到了计算机学院的信息安全专业。

大二时我接触到了 CTF(Catch the Flag)，还加入了学院的 CTF 战队，碰到了许多大佬和师傅，接触到了一个和以前完全不同的群体。大二寒假时，我加入了学院的组织的科创比赛团队，第一次接触到了 iOS 开发。那时的 Swift 语言还没有完全成形，正是一个推荐使用 Swift 但大多数公司还在使用 Objective-C 的时期。拖动组件，调节布局，编写组件行为这些步骤我还记忆犹新。

大三时我决定考研，当时学校建议考研的同学参加学院老师提供的暑假实习，再加上和学院同学的信息差，我便没有去找企业实习，而是在实验室进行了三个月的实习。这段实习的主要任务是为实验室后续研究搭建 Hbase，在网络中用 LXD、Quagga 搭建了一个路由环境，用 OpenTSDB 分析流量，这也是我第一次接触到分布式环境搭建，当时出现了很多莫明其妙的 bug，我和小组的几个成员翻遍了 google，面对着漆黑的 cli 艰难前进。这段经历也反过来提升了我对于 Web 前端的兴趣，交互和所见即所得是我热爱前端的两个原因。

大一时我就仿效大佬们在博客园写博客，后来切换到了 Hexo，在腾讯云买了服务器和域名，建立了第一个网站(starfurye.space，部分文章已迁移过来)，后来准备考研也几乎没有再更新了。Hexo 给我最大的感受就是可定制性太差了，我只能对着 yaml 的配置文件改改参数，换个图片，除此以外就没有能做的事了。受此影响，我在研究生录取安定下来后开始学习 html+css+js，但当时也是粗浅的学学，还在[Udemy](https://udemy.com)上面买课，坚持下来的也只有用 js 写 leetcode，差不多写了半年多，js 红宝书也只看了一遍，里面的高级用法由于使用场景的匮乏也是看了就忘。

直到 2021 年底我才了解到前端求职原来要会那么多东西，2022 年 2 月才开始跟着课程做[拾趣](https://shiqv.vercel.app)这个全栈项目，这个博客也是在我会用基本的 React 后才着手编写的，每个组件、样式表都完全受我掌控，对这个项目的自由度让我非常满意。
